## 🎨 HTML 特点

- **商务风格配色**：深蓝/青色渐变背景
- **响应式设计**：支持手机/平板/电脑
- **重要性标识**：🔥 热门（8-10 分）、⭐ 推荐（6-7 分）、📰 普通（<6 分）
- **悬停效果**：鼠标悬停卡片右移，阴影增强
- **来源标签**：清晰标注新闻来源和类别
- **阅读原文**：每条都有直达链接

## ⚙️ 配置说明

### 修改关键词

编辑 `skill_v5.py` 中的 `AI_KEYWORDS` 和 `HOT_KEYWORDS` 列表。

### 修改数据源

在 `AIDailyScraper` 类中添加新的 `fetch_xxx()` 方法。

### 修改样式

编辑 `generate_html()` 函数中的 CSS 样式。
